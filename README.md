### Halo dunia! 👋 Saya Reinhard Batubara

Saya adalah seorang mahasiswa Sistem Informasi di Institut Teknologi Del yang berfokus pada **Data Science**, **Deep Learning**, dan **Full-Stack Web Development**. Saya memiliki kegemaran kuat dalam memecahkan masalah kompleks, merancang arsitektur database yang efisien, serta membangun solusi berbasis kecerdasan buatan untuk mengoptimalkan sistem perangkat lunak.

Saat ini, saya sedang mengembangkan sistem klasifikasi motif berbasis *Deep Learning* menggunakan *Convolutional Neural Network* (MobileNetV2) yang diintegrasikan dengan *Retrieval-Augmented Generation* (RAG) untuk mendukung sistem edukasi secara interaktif dan kontekstual.

* **Framework & Web:** Laravel, Django, CodeIgniter, EasyUI
* **Desain & Arsitektur Database:** ERD, PDM, Normalisasi, Optimasi Kueri SQL
* **Alat Pengembangan & Otomatisasi:** Git, GitHub, Postman, Figma, n8n, Trello, Asana

#### 💼 Pengalaman Proyek Pilihan

* **Sistem Klasifikasi Motif & Edukasi Budaya Berbasis RAG (Tugas Akhir):** Mengembangkan arsitektur *Deep Learning* untuk klasifikasi objek visual dikombinasikan dengan *pipeline* RAG menggunakan *vector database* dan LLM untuk ekstraksi informasi yang akurat.
* **DelDeals (Platform E-Commerce Kampus):** Bertindak sebagai *Full-Stack Developer* dalam merancang dan membangun platform jual-beli internal mahasiswa berbasis Laravel dan MySQL dalam siklus pengembangan tangkas (*Agile*).
* **Sistem Web ERP - Ebako:** Mengikuti magang sebagai *Programmer Intern* untuk membangun, memperbarui, dan melakukan penanganan kesalahan (*debugging*) pada modul operasional bisnis skala enterprise.
* **Situs Web Komunitas (HKBP Sigumpar & Madrasah):** Mentransformasikan kebutuhan non-teknis menjadi fitur fungsional web yang stabil untuk mendukung diseminasi informasi dan administrasi digital.
